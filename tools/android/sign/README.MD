# android apk签名步骤

* 运行文件sign.bat。根据提示选择相关文件。

# 注意事项

* 若是当前ionic项目，可直接选择默认选项，会在当前目录中生成签名后的release.apk。过程中需要输入默认秘钥的密码，密码可根据提示信息输入。
* 签名需要jdk以及android环境。
